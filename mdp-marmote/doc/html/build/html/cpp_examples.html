C++ Tutorial and ExamplesÔÉÅ

The following examples show elementary manipulations of Marmote objects:

  • Example 1: create a Markov Chain and perform Monte-Carlo simulation

  • Example 2: create a Markov Chain and compute transient distributions

  • Example 3: as in Example 2 with several chains

  • Example 4: read and write Markov Chains from/to files

  • Example 5: create a Markov Chain and compute the stationary distribution with different methods

  • Example 6: manipulate Sets (state spaces)

  • Example 7: create Markov Chains and perform structural analysis

The following examples show how to use more advanced Marmote objects:

  • Example 10: use the multidimensional random walk objects

The following examples to manipulate Markov Decision Process are provided:

  • ExampleMDP 10: create a discounted MDP object and describes the generalities about the MDP modelling. It also contains a short presentation of the MDP solver.

  • ExampleMDP 11: create a discounted MDP and illustrate the way to get the same action set for any state.

  • ExampleMDP 20: create an average MDP.

  • ExampleMDP 21: create an average MDP and illustrate the way to evaluate a policy.

  • ExampleMDP 22: create an average MDP. In this example we create the MDP before and add the matrices after.

  • ExampleMDP 30: create an total Cost MDP for computing a toy Stochastic Shortest Path.

  • ExampleMDP 31: create an total Cost MDP for computing the well known SSP four room and illustrate how to manipulate a multidimensionnal state space.

  • ExampleMDP 40: create a finite horizon MDP given by Puterman in his book.
