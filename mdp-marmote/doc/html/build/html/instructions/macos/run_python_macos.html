Using the Python API of Marmote on MacOS

Examples provided are in two forms: Python notebooks and simple Python program files.

Preparation

Prepare an installation directory, and move into it. We assume throughout that MAR_DIR is the path to this directory.

$ mkdir MAR_DIR
$ cd MAR_DIR

Running the notebooks

Download the notebooks files: as zip file to the directory MAR_DIR. Unpack the source files:

$ unzip all_notebooks.zip

Then move in the directory and type

$ jupyter-notebook name_of_notebook.ipynb

If the installation of jupyter is required then type

$ conda install -c conda-forge jupyter

Running Python files

Download all the Python files: as zip file to the directory MAR_DIR. Unpack the source files:

$ unzip all_pythons.zip

Then move in the directory and type

$ python name_of_script.py
